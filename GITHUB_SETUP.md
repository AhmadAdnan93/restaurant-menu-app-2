# GitHub Setup Instructions

## Quick Push Commands

After creating the repository on GitHub, run these commands:

### Push Main Branch:
```bash
git push -u origin main
```

### Push Develop Branch:
```bash
git push -u origin develop
```

## Branches Created:
- `main` - Production-ready code
- `develop` - Development branch for your team

## Share with Developers:
1. Give them the repository URL: `https://github.com/AhmadSulieman93/restaurant-menu-app`
2. They can clone it with:
   ```bash
   git clone https://github.com/AhmadSulieman93/restaurant-menu-app.git
   cd restaurant-menu-app
   npm install
   npm run dev
   ```

## Working with Branches:
- Switch to develop: `git checkout develop`
- Create new branch: `git checkout -b feature/new-feature`
- Push branch: `git push -u origin feature/new-feature`
